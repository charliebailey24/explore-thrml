# explore-thrml
Sandbox to play around with Extropics' new THRML Python Library.

## About THRML

THRML is a Python library from Extropics for thermodynamic computing. This sandbox provides a simple environment to explore and experiment with the library.

## Setup

### 1. Create a virtual environment (recommended)

```bash
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

## Usage

### Running the basic example

```bash
python basic_example.py
```

### Interactive exploration with Jupyter

Start a Jupyter notebook server:

```bash
jupyter notebook
```

Or use IPython for interactive exploration:

```bash
ipython
```

Then import and explore THRML:

```python
import thrml
# Explore the library here
```

## Examples

- `basic_example.py` - A simple script demonstrating basic THRML usage
- `advanced_example.py` - More detailed exploration of THRML components
- `explore_thrml.ipynb` - Interactive Jupyter notebook for hands-on exploration
